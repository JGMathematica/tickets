这是一个[Wox](https://github.com/Wox-launcher/Wox)插件，用来查火车票信息。

## 基本功能：

1. 基本的查火车票的功能：

   `tk 南京 上海 201674`

   ![1](preview\1.jpg)

   可以接受的日期格式有：201674, 20160704, 2016-7-4, 2016/7/4等。

2. 可以添加参数来查询具体车次：

   `tk 南京 上海 201674 D ` ![2](preview\2.jpg)

   ​

   `tk 南京 上海 201674 D31`

   ![3](preview\3.jpg)

   ​

   `tk 南京 上海 201674 D3125` ![4](preview\4.jpg)

3. 帮助信息：

   `tk -h`
